# Student Performance Analysis

## Description
This project analyzes factors affecting student performance and predicts exam scores using machine learning.

## Dataset
- Features: study hours, attendance, sleep hours, previous scores

## Steps Performed
1. Data Cleaning
2. Exploratory Data Analysis
3. Visualization
4. Model Building (Linear Regression)
5. Evaluation

## Results
- Identified key factors affecting performance
- Built a predictive model

## Tools Used
- Python
- Pandas, Matplotlib, Seaborn, Scikit-learn

## Conclusion
The model helps understand how different factors influence student scores.
