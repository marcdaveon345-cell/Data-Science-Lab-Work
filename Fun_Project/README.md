# AI Productivity Dashboard

## Description
This project is a personal productivity dashboard that analyzes how time is spent across different activities.

## Dataset
- Sample dataset with tasks, categories, time spent, and dates

## Steps Performed
1. Data Cleaning
2. Data Analysis
3. Visualization (bar, line, pie charts)
4. Prediction using Linear Regression

## Results
- Identified time usage patterns
- Visualized productivity trends

## Tools Used
- Python
- Pandas, Matplotlib, Seaborn, Scikit-learn

## Conclusion
The dashboard helps understand productivity habits and improve time management.
